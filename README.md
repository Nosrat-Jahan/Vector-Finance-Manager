# 💰 Vector-Finance-Manager (v1.0.1)

**Vector-Finance-Manager** is a legacy financial intelligence module designed to automate personal wealth management. Originally architected in **October 2025**, it features real-time transaction tracking and a floating AI assistant.

## 📁 Project Architecture
- **`src/pages/`**: Contains the core application logic (`Index.tsx`).
- **`src/components/`**: UI modules for Dashboard and AI Assistant.
- **`src/hooks/`**: Custom React hooks for authentication and state sync.
- **`src/lib/`**: Database integration and logic layer for `financeDb`.

## 🛠️ Tech Stack
- **Frontend:** React & TypeScript
- **Styling:** Tailwind CSS & Lucide Icons
- **State Management:** Custom Hooks & Context API

## 👨‍💻 Installation
1. `git clone https://github.com/Nosrat-Jahan/Vector-Finance-Manager.git`
2. `npm install`
3. `npm run dev`
